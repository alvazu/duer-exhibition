Duer.org - An Open-Source C-Like Programming Language

Duer(originally named "C--") is an open-source C-Like Programming Language, which makes C into a flexible language.

Duer Builder(originally named "C-- builder") is a Duer interpreter.

Duer Builder 0.80 is a Public-domain software.
